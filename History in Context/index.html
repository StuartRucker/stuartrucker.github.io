History in Context Java, HTML, CSS, Javascript

Did you know that the Aztec empire existed at the same time as Oxford? History in Context does!

Since history is often taught only with respect to one civilization, people are often unaware that multiple events happened at similiar time period. History in Context aims to provide a visual way learn about time periods.

History in Context has a front end website which displays a time line. User's can enter any event or person. This entry is sent to a java server which scrapes wikipedia, and looks for the date. However, because wikipedia is written by many different users, there is not a standard place to find dates. Therefore, occasionally the date found during scraping is not correct.
