Internet Monitored Sprinkler

Arduino, HTML, Python, Javascript, CSS, Firebase

This IOT sprinkler system incorporates a moisture sensor, and valve control over the whole yard. Never again will water be wasted after a rainstorm, when we are not there to turn off the sprinkler.

History in Context

Java, Javascript, HTML, CSS

Did you know that the Aztec empire existed at the same time as Oxford. This website allows users to enter any person or event in history. The website scrapes wikipedia and assembles a timeline, so the user can see how the dates of different events compare.

Othello with AI

Java

This computer program allows you to play the classic board game Othello agianst the computer's Artificial Intelligence. The AI makes a tree of potential future gamestates and evaluates what the best move is using Minimax.

Exonian Email Newletter

HTML, CSS, Java

It is often hard to remeber when a new issue of The Exonian, Phillips Exeter's online newspaper, comes out. This project allows users to subsribe to an email newsetter, with highlights and links to the latest issue

  • A java program to generate said HTML

Arduino Clock

Arduino, Adobe Illustrator, Laser-Cutter

This laser cut, Arduino powered clock provides has an aesthetically pleasing laser cut frame. It icorporates both anilog and digital features with 12 LED's lining the perimeter to denote hours, and a rotating wheel in the center to denote seconds. Using the control panel, the time can be adjusted, and an alarm can set.
